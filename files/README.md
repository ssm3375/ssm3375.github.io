# Files Directory

Add your personal files here:

## Profile Photo
Add your professional profile photo as `profile.jpg` (or `profile.png`).

**Recommended specifications:**
- Size: 400x400 pixels minimum
- Format: JPG or PNG
- Professional headshot with good lighting

The photo will be displayed as a circular image (180px diameter).

## CV/Resume
Add your CV as `cv.pdf` for the download button to work.

**Note:** If these files are missing, the site will show placeholders:
- Photo: Shows "SM" initials placeholder
- CV button: Will still appear but won't download anything until you add the file
